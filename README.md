# Construction Management Software

A comprehensive construction project management system designed specifically for the Pakistani construction industry workflow. This software helps construction companies track projects from land acquisition to final handover, managing all phases including planning, foundation work, construction, and finishing.

## üèóÔ∏è Features

### Core Project Management
- **Project Lifecycle Tracking**: Complete workflow from land acquisition to handover
- **Phase Management**: Planning, Foundation, Construction, Elevation phases
- **Floor-by-Floor Progress**: Track grey structure and finishing work for each floor
- **Milestone Tracking**: Monitor key construction milestones and approvals

### Construction Workflow (Pakistani Standard)
1. **Planning & Feasibility**
   - Land survey and soil testing
   - Architectural planning and approvals
   - Sale NOC acquisition
   - Legal compliance tracking

2. **Foundation Work**
   - Piling work (if required)
   - Raft foundation
   - Plinth construction
   - Plinth verification

3. **Construction Phase**
   - Grey structure (floor by floor)
   - Finishing work (parallel to grey structure)
   - Quality control and inspections
   - Engineer approvals

4. **Final Work**
   - Elevation finishing
   - Final inspections
   - Project handover

### Task Management
- **Task Assignment**: Assign tasks to engineers, contractors, and teams
- **Progress Tracking**: Monitor task completion and progress
- **Dependencies**: Track task dependencies and critical paths
- **Priority Management**: High, medium, low priority classification

### Quality Control
- **Cube Tests**: Track concrete strength tests
- **Engineer Inspections**: Monitor inspection schedules and results
- **Material Testing**: Quality verification for steel, cement, etc.
- **Compliance**: Ensure adherence to Pakistani building codes

### Resource Management
- **Human Resources**: Engineers, contractors, laborers
- **Equipment**: Heavy machinery, tools, and vehicles
- **Materials**: Inventory tracking and procurement
- **Utilization**: Monitor resource allocation and efficiency

### Financial Tracking
- **Budget Management**: Track project budgets and expenditures
- **Cost Breakdown**: Land, approvals, foundation, construction, finishing
- **Expense Tracking**: Monitor actual vs. planned costs
- **Financial Reports**: Generate comprehensive financial summaries

### Reporting & Analytics
- **Progress Reports**: Visual progress tracking and analytics
- **Performance Metrics**: KPI dashboards and trend analysis
- **Quality Metrics**: Pass/fail rates and compliance tracking
- **Resource Utilization**: Efficiency and allocation reports

## üöÄ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm (v8 or higher)

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd construction-software
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Open your browser**

## üèóÔ∏è Construction Workflow Usage

### 1. Project Creation
1. Navigate to Projects ‚Üí Add New Project
2. Fill in project details (name, location, client, budget)
3. Set project timeline and total floors
4. Assign architect, engineer, and contractor

### 2. Planning Phase
1. Update project status to "Planning & Feasibility"
2. Track land survey completion
3. Record soil testing results
4. Monitor architectural plan approval
5. Track Sale NOC acquisition

### 3. Foundation Phase
1. Update project status to "Foundation Work"
2. Track piling work progress
3. Monitor raft foundation completion
4. Record plinth construction
5. Document plinth verification approval

### 4. Construction Phase
1. Update project status to "Construction"
2. Track grey structure progress floor by floor
3. Monitor finishing work for completed floors
4. Record quality control inspections
5. Document engineer approvals

### 5. Final Phase
1. Update project status to "Elevation & Final Work"
2. Track elevation finishing
3. Monitor final inspections
4. Document project handover
